# Support

Use this repository for public Waspen release and installer support.

## Use GitHub Issues For

- Broken GitHub or Backblaze download links
- Checksum mismatches
- Installer launch failures
- Public beta release regressions
- Documentation gaps in release notes

## Email Support For

- License generation or license recovery
- Private diagnostics
- Enterprise or fleet deployment questions
- Security-sensitive reports

Email: support@alphabytez.dev

## Before Opening an Issue

Include your OS version, CPU architecture, Waspen version, installer filename, and whether the download came from GitHub Releases or a Backblaze mirror.
